# Expense Tracker in Python

## Project Overview
This is a simple Expense Tracker developed using Python. The application allows users to add, view, and calculate expenses.

## Features
- Add expenses
- View expense records
- Calculate total expense
- Simple menu-driven interface

## Technologies Used
- Python

## How to Run
1. Open the project in VS Code
2. Run the Python file
3. Add and view expenses
4. Check total expense
5. Exit the program

## Project Outcome
This project helped in understanding Python loops, dictionaries, user input handling, and calculations.
